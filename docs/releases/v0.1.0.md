# Release Notes v0.1.0

`workledger` v0.1.0 is the first public release of the trace-to-work layer.

## Summary

This release makes the project publicly usable from source with the core local-first ledger model, ingestion pipeline, rollup engine, policy-backed classification flow, report outputs, and docs.

The important story for this release is:

- raw traces become reviewable `WorkUnit`s
- evidence and uncertainty stay attached
- review queues remain visible instead of being flattened away
- comparative economics remain explicit and editable
- downstream publication bundles can extend from the same canonical work layer

## Quick Install And Try

PyPI distribution is not live for this release. Install from source from this repository:

```bash
git clone https://github.com/couscous18/workledger.git
cd workledger
uv sync --all-extras
uv run wl demo open-traces --project-dir .workledger/open-traces --open-report
uv run wl compare-costs --from-project .workledger/open-traces
```

## What Ships In v0.1.0

- Core models for `ObservationSpan`, `WorkUnit`, `ClassificationTrace`, `PolicyDecision`, `EvidenceRef`, `PolicyPack`, and `ReportArtifact`
- Ingestion for JSONL, OpenInference-like payloads, OTEL JSON, CloudEvents JSON, and SDK-shaped events
- Rollup that compresses low-level spans into business-level work units
- Declarative policy packs with explainable rule matches and review-required states
- DuckDB-backed local store with JSON, CSV, and Parquet export plus terminal, Markdown, JSON, CSV, Parquet, and HTML reports
- `wl` CLI, local FastAPI server, runnable demos, JSON Schema export, and published OpenAPI artifacts
- A downstream software capex publication bundle that shows how trace-to-work can support Hugging Face-style OSS packaging

## Alpha Status

`workledger` v0.1.0 is an alpha release. It is intended to make the project publicly usable and reviewable, not to promise stable interfaces yet.

## Known Limitations

- Alpha quality: rough edges are expected
- Breaking changes may happen before a stable `1.0.0`
- Local or source install is required until package distribution is live
