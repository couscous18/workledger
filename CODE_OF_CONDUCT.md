# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the full text at the link above. By participating in this project, you agree to abide by its terms.

## Reporting

If you experience or witness unacceptable behavior, use a GitHub-native path that the repository can actually monitor:

- Open a GitHub Discussion or issue for routine moderation concerns when you are comfortable doing so publicly.
- If the situation should not be discussed in public, contact a repository maintainer through GitHub and keep public details minimal.

This project is community-maintained, so moderation is best effort rather than SLA-backed. Reports will be handled with care and privacy when possible.
